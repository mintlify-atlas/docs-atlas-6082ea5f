# F1 Stats Archive Documentation

This repository contains the documentation for the F1 Stats Archive project - a comprehensive Formula 1 historical statistics archive with automated data collection from the Ergast API.

## About the Project

F1 Stats Archive collects and organizes Formula 1 race data from 1950 to the present, including:
- Race results and qualifying sessions
- Driver and constructor championship standings
- Lap times and pitstop data
- Sprint race results

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview the documentation locally:

```bash
npm i -g mint
```

Run the following command at the root of the documentation:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.
